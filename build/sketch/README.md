# smart-car
Проект представляет собой систему управления роботом на базе микроконтроллера с использованием компьютерного зрения. Библиотека Python MediaPipe используется для отслеживания положения руки, а обученная модель машинного обучения распознает жесты и преобразует их в команды управления устройством.

## суть работы проекта
1. Камера захватывает изображение в реальном времени.
2. MediaPipe определяет положение руки и пальцев.
3. Обученная модель машинного обучения распознает жест.
4. Python отправляет команду на устройство.
5. Устройство(машинка) выполняет движение.

## Используемые технологии
1. Python
2. Машинное обучение
3. Устройство с микроконтроллером ESP32

## Установка
```bash
git clone https://github.com/margosha-soso/smart-car
cd smart-car
pip install -r requirements.txt
```

## Запуск проекта
```bash
python main.py
```
Перед запуском убедитесь, что:
- подключена камера
- загружена обученная модель
- включено устройство управления
